# 更新日志 (CHANGELOG)

## [1.4.9] - 2026-03-18

### 🛑 真正的运行时停止 (True Backend Stop)

- **后端直通中断 (Stop API)**: 现在在预处理运行时点击 toastr 通知，会真正触发酒馆后端的 `stopGeneration` 请求，不再只是 UI 维度的本地状态变更。
- **多级 Fallback 策略**: 实现了从酒馆 Context API -> 全局 `stopGeneration` -> 模拟 `#mes_stop` 按钮点击的层级后备逻辑，确保中控可靠性。

### 📊 Dashboard 稳定性与性能修复 (P0)

- **数据库 V3 升级**: 为 `events` 和 `entities` 表新增了 `is_archived` 和 `is_embedded` 索引。
- **RAG 检索性能飞跃**: `Retriever` 查询逻辑重构，大幅减少了不必要的全表扫描深度。
- **功能开关同步修正**: 修复了 Dashboard feature toggles 与持久化配置在刷新后可能出现的类型显示偏移。

### 💄 UI/UX 细节体验优化

- **MemoryStream 管理增强**:
  - **快速操作栏**: 在列表卡片上直接暴露“归档、锁定、删除”操作，减少跳转。
  - **编辑器对齐**: 在详情页头部增加了“锁定”与“归档”切换按钮，使深度编辑更连贯。
- **API 预设视觉引导**: 强化了正则规则列表中“新增规则”按钮的权重与可见度，降低操作发现成本。

### 📝 正则规则精进

- **模型思考适配**: `think` 清洗规则现在支持 `<thinking>` 标签及其多种变体。
- **反截断增强**: 新增了针对 `<disclaimer>` 标签及其内容的自动清洗规则。

## [1.4.8] - 2026-03-17

### 🚀 Dashboard 性能与稳定性增强 (Dashboard Performance & Stability)

- **UI 节流优化 (Log Throttling)**: 为仪表盘日志流引入了 500ms 的更新节流逻辑，彻底解决高频日志产出时导致的界面卡顿与渲染过载。
- **状态同步修复 (Stale Closure Fix)**: 修复了仪表盘可见性切换时的闭包过期问题，确保数据刷新逻辑始终指向最新实例。
- **并发安全控制 (Race Condition Fix)**: 重构了功能开关的异步链路，解决了快速连点导致的竞态条件隐患。
- **统计加载优化**: 增加了防御性错误处理并优化了大规模记忆数据的统计逻辑，提升首屏加载速度。

### 🔧 遗留问题修复与功能补全 (Bug Fixes & Feature Polish)

- **API 认证补全 (CSRF Security)**: 为更新检测服务补全了 CSRF 请求头，解决了在新版酒馆下请求 `/api/extensions/version` 报 404/401 的认证问题。
- **记忆编辑器归档开关**: 在「记忆流」事件编辑器中增加了备受期待的“手动归档 (Archive)”开关，允许用户更精准地控制 RAG 扫描范围。
- **提取超时容忍度提升**: 将实体提取的 watchdog 超时上限提升至 **30 分钟**，完美适配长上下文及低速本地模型。
- **日志噪音降低**: 将正则清洗产生的预期内空消息警告降级为 `debug` 模式，清理了控制台刷屏。

### 🛡️ 类型系统与代码健壮性 (Type Safety)

- **全局类型统一**: 集中化处理了 `SillyTavern` 与 `selected_model` 的 Window 接口声明，消除了重复定义导致的类型冲突。
- **异常边界加固**: 修复了世界书扫描逻辑中的潜在空指针风险，提升了极端场景下的系统稳定性。

## [1.4.7] - 2026-03-13

### ✨ 交互性与性能优化 (Interaction & Performance)

- **手动记忆/实体创建 (Manual Creation)**:
  - 在「记忆流」ActionBar 中新增了 ➕ "添加" 按钮。
  - 支持直接手动创建空白事件或实体记录，数据即时落库并自动打开编辑器。
- **全局动画开关 (Global Animation Toggle)**:
  - 在「设置」-「外观」中新增了“启用 UI 动画”全局开关。
  - 开启时保留极致视觉体验（GSAP 入场/出场帷幕、Framer Motion 页面切页、CSS 微动画）。
  - 关闭时将禁用所有补间动画与过度，大幅提升低配设备或追求极速操作用户的响应速度。
- **移动端交互修复 (Mobile UX Fixes)**:
  - 修复了移动端在全屏编辑器状态下由于 Tab 遮挡导致无法看到“保存”按钮的 UI Bug。
  - 现在移动端编辑器头部会动态显示 💾 保存操作，确保修改可持久化。

### 💄 动画视觉细节打磨

- **开场重构**: 将拉幕动画逻辑移入 `MainLayout`，通过 `useState` 锚定单次开启内的渲染方向，彻底解决了偶发的连续两次播放动画的 Bug。
- **色调调整**: 将帷幕终点色由强调色 (`primary`) 改为背景色 (`background`)，使入场衔接更平滑，尤其是在深色主题下大幅降低了视觉冲击感。
