# 内置帮助文档编写指南

本项目使用 **MDX** (Markdown + JSX) 编写内置帮助文档，支持在 Markdown 中直接嵌入 React 组件。

---

## 文件结构

```
src/docs/
├── types.ts              # 类型定义
├── index.ts              # 文档注册表 ⭐ 添加新文档需修改
├── getting-started.mdx   # 快速入门文档
└── [新文档].mdx          # 你的新文档
```

---

## 添加新文档的步骤

### 1. 创建 MDX 文件

在 `src/docs/` 目录下创建 `.mdx` 文件：

```mdx
{/* src/docs/configuration.mdx */}
import { Settings, Database } from 'lucide-react';

# 配置指南

## LLM 配置

点击 <Settings className="inline-block w-4 h-4 align-text-bottom" /> **设置** 进行配置...
```

### 2. 注册文档

在 `src/docs/index.ts` 中添加引用：

```typescript
// 1. 导入 MDX 组件
import Configuration from './configuration.mdx';

// 2. 添加到 DOCS 数组
export const DOCS: DocCategory[] = [
    // ... 现有文档
    {
        id: 'configuration',
        label: '配置指南',
        icon: Settings,           // Lucide 图标
        component: Configuration, // MDX 组件
        keywords: ['设置', 'config', 'LLM', 'API'],
    },
];
```

### 3. 构建验证

```bash
npm run build
```

## MDX 语法速查

### 嵌入图标

```mdx
import { Settings, Brain } from 'lucide-react';

点击 <Settings className="inline-block w-4 h-4 align-text-bottom" /> 设置
```

### 提示块

```mdx
> **提示**：推荐使用 OpenAI 兼容 API

> [!WARNING]
> 这是一个警告
```

### 代码块

````mdx
```typescript
const example = 'hello';
```
````

---

## 注意事项

1. **图标样式**：使用 `className="inline-block w-4 h-4 align-text-bottom"` 保持图标与文字对齐
2. **关键词**：在 `keywords` 数组中添加搜索关键词，支持全局搜索
3. **Tab 顺序**：文档在 Tab 中的顺序由 `DOCS` 数组顺序决定
