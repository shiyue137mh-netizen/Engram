# 常见问题 (FAQ)

---

### Q: 自动总结没有触发？

触发条件检查：
1.  **楼层间隔**: 新消息数量未达到设定的阈值（默认为 5-10 条，Dashboard 可调）。
2.  **串行保护**: 上一轮总结任务仍在进行中。

### Q: 开启 RAG 后响应变慢？

RAG 增加了网络请求环节（Embedding/Rerank API）。
*   **建议**: 使用本地 `Transformers.js` 向量源可消除此延迟。

### Q: API 报错 (404/Connection Refused)？

*   检查 API 地址是否填写正确（通常需包含 `/v1`）。
*    CORS 问题：确保您的 API 服务端允许跨域请求（如从 `localhost` 发起）。

### Q: 修改摘要后搜不到？

修改摘要文本后，旧的向量数据失效。
*   **操作**: 在记忆编辑页面点击 **Re-embed** 按钮重建索引。

### Q: 数据备份？

在 **数据处理** 页面导出 JSON 文件。数据仅存储在本地浏览器 IndexedDB，请定期备份以防浏览器缓存清除。
